# 🍎 Calorie Tracker JS

A simple **Calorie Tracking Web App** built using **HTML, CSS, and JavaScript**.  
This project helps users calculate their daily calorie intake from a wide variety of foods including **Pakistani dishes, dry fruits, drinks, and international fast foods**.

---

## 🚀 Features
- ✅ Add foods and track calories in real-time  
- ✅ Pre-loaded food database with **Pakistani + International dishes**  
- ✅ Dry fruits, snacks, drinks, and dairy included  
- ✅ Responsive design (works on mobile & desktop)  
- ✅ Lightweight and **no backend required** (only HTML/CSS/JS)

---

## 📦 Food Categories
- 🥭 Fruits (Banana, Apple, Mango, etc.)  
- 🌰 Dry Fruits (Almonds, Walnuts, Cashews, etc.)  
- 🥛 Dairy (Milk, Yogurt, Paneer, etc.)  
- 🍵 Drinks (Tea, Green Tea, Coffee, Soft Drinks, etc.)  
- 🍛 Pakistani Dishes (Biryani, Karahi, Haleem, Nihari, etc.)  
- 🍔 Snacks & Fast Food (Pakora, Samosa, Burger, Pizza, etc.)

---

## 🛠️ Tech Stack
- **HTML5**  
- **CSS3**  
- **JavaScript (Vanilla)**
